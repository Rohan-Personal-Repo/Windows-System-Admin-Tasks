# PowerShell-Scripts

A handy repository for storing and sharing useful PowerShell scripts. This collection helps system administrators, DevOps engineers, and Windows power users automate tasks, manage systems, and streamline workflows.

---

## Overview

Find a variety of scripts to help with automation, file and system management, reporting, and other day-to-day operations. Each script is well-commented and ready for implementation or adaptation.

---

## Contents

- System information gathering
- File and directory manipulation
- User account management
- Windows service management
- READ CSV File
- Various utility tools and functions

---

## PowerShell Execution Policy

For PowerShell scripting, if you encounter an issue with being unable to run certain scripts, please change the execution policy.  
See details here:  
- [Windows-System-Admin-Tasks: PowerShell Scripting Execution Policy](https://learn.microsoft.com/en-us/powershell/module/microsoft.powershell.core/about/about_execution_policies?view=powershell-7.5)

Example command to change execution policy (run as Administrator):

**Set-ExecutionPolicy RemoteSigned**

## Getting Started

1. **Clone the repository**

git clone https://github.com/Rohan-Repo/PowerShell-Scripts.git

2. **Open any `.ps1` file in Windows PowerShell, PowerShell Core, or VS Code**
3. **Customize variables and parameters as needed**
4. **Run scripts from the command line**

---

## Prerequisites

- Windows PowerShell 5.x or PowerShell Core 7.x+
- Administrator privileges (for some system-level scripts)
